# 🚀 30 Days Python Challenge

> A structured collection of problems solved to strengthen Python fundamentals, logic building, and problem-solving skills.

---

## 🟢 Beginner Level (Q1–Q10)

### 1. Even or Odd

Determine whether a number is even or odd.

### 2. Largest of Three Numbers

Find the largest among three input numbers.

### 3. Sum of First N Natural Numbers

Calculate the sum from `1` to `n`.

### 4. Reverse a String

Reverse a given string.
**Example:** `"hello" → "olleh"`

### 5. Count Vowels

Count vowels in a given string.

### 6. Factorial of a Number

Compute factorial using loops.
**Example:** `5 → 120`

### 7. Prime Number Check

Check whether a number is prime.

### 8. Simple Calculator

Perform basic arithmetic operations: `+`, `-`, `*`, `/`.

### 9. Count Digits

Count total digits in a number.
**Example:** `12345 → 5`

### 10. Maximum in a List

Find the maximum element without using built-in functions.

---

## 🟡 Intermediate Level (Q11–Q20)

### 11. Palindrome Number

Check if a number is a palindrome.
**Example:** `121 → Yes`, `123 → No`

### 12. Palindrome String

Check whether a string is a palindrome.

### 13. Fibonacci Series

Print first `n` Fibonacci numbers.
**Example:** `0 1 1 2 3 5`

### 14. Word Count

Count words using `.split()`.

### 15. Second Largest Element

Find the second largest number without `sort()` or `max()`.

### 16. Remove Duplicates

Remove duplicates from a list (logic or set).

### 17. Character Frequency Counter

Count frequency of characters using a dictionary.

### 18. Armstrong Number

Check if a number is an Armstrong number.
**Example:** `153 → Valid`

### 19. Manual Sorting

Sort a list using Bubble Sort / Selection Sort.

### 20. Guess the Number Game

Build a guessing game using `random`:

* Too high
* Too low
* Correct

---

## 🔴 Advanced Level (Q21–Q30)

### 21. Anagram Checker

Check if two strings are anagrams.
**Example:** `"listen" & "silent" → Yes`

### 22. Longest Word in Sentence

Find the longest word in a sentence.

### 23. Matrix Addition

Add two matrices using nested loops.

### 24. Matrix Transpose

Compute matrix transpose.

### 25. Remove Punctuation

Remove punctuation from a string.

### 26. List Rotation Check

Check if one list is a rotation of another.

### 27. Binary Search

Implement binary search on a sorted list.

### 28. Prime Numbers in Range

Print primes within a given range.

### 29. Password Strength Checker

Validate password based on:

* Minimum length (8)
* Uppercase letter
* Lowercase letter
* Digit
* Special character

### 30. Mini Contact Book (CRUD)

Build a contact system using a dictionary:

* Add
* Search
* Delete
* Display
